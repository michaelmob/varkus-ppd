Embedding Your Widget

Embedding your Widget is very straight forward. Some knowledge of HTML, CSS, and Javascript would be beneficial but is not needed to embed a basic Widget.

Viral Widget Visitor URL/ID Option

The way viral widgets transfer the Visitor ID is through the URL fragment of the Visitor URL. You may want to change the Visitor URL to include the Visitor ID in a different way, this is possible and simple to do.

Add a "url" attribute to the "iframe" tag. Place an "{id}" where you would like the Visitor ID to be placed within the URL. When doing this, you must also create a Javascript variable called "id_{{ object.code }}" (notice after "id_" is the code of the widget) with the visitor ID that is found within the URL.
