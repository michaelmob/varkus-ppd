Using this form you can submit an abuse report to us. We dislike rule breaking/illegal content just as much as you do. We thoroughly investigate each report to make sure that an innocent user does not get punished. So, please do not submit false abuse reports.

Instructions

When creating an abuse report please make sure you:

  • Enter a valid e-mail, as we will e-mail you if action is taken towards the content you reported.

  • Provide a valid reason. We cannot punish a user or remove content if you put in just one or two words.

  • Provide evidence. A link to a website or a blog that might be taken down quickly does not help. An archived page or a screenshot is much more helpful.
